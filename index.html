Quais ferramentas eu usei para criar este Landing Pages ?

HTML: A Estrutura

​O HTML (HyperText Markup Language) é a linguagem de marcação que define a estrutura e o conteúdo de uma página web. Imagine-o como o esqueleto de um prédio: ele determina onde ficam as paredes, as portas e as janelas, mas não cuida da pintura ou da fiação elétrica.
​Através de tags (como <h1>, <p>, <a> e <img>), o HTML organiza os elementos para que o navegador saiba o que é um título, um parágrafo ou um link. Sem o HTML, a internet seria apenas um amontoado de texto sem ordem ou hierarquia.

CSS: O Estilo

O CSS (Cascading Style Sheets) é o que traz vida visual ao projeto. Se o HTML é o esqueleto, o CSS é a pintura, o revestimento e a decoração. Ele é responsável por definir cores, fontes, tamanhos, espaçamentos e o layout (disposição dos elementos) na tela. ​Uma das maiores vantagens do CSS é a separação de conceitos: você pode mudar completamente o visual de um site inteiro editando apenas um arquivo CSS, sem precisar mexer no conteúdo estrutural do HTML. É ele quem garante que o seu site seja bonito e, principalmente, responsivo (que se adapta a celulares e computadores).

JavaScript: O Comportamento

O JavaScript (JS) é a linguagem de programação que adiciona interatividade e dinamismo. Enquanto o HTML estrutura e o CSS embeleza, o JavaScript faz as coisas "acontecerem". ​É o JS que permite que você clique em um botão e abra um menu lateral, valide um formulário de cadastro em tempo real, ou atualize o feed das suas redes sociais sem precisar recarregar a página. Ele transforma documentos estáticos em aplicações web complexas e inteligentes, sendo hoje uma das linguagens mais poderosas e populares do mundo.
